Welcome

This Java web application you is a phone book application with contact management and note-taking functionalities. Here is a synopsis of the application's features:
User Registration: Users can create an account to access the application's features. They will provide their personal details such as username, password, and possibly additional information.
User Authentication: After registration, users can log in to the application using their credentials. This ensures that each user can access their own contacts and notes securely.
Contact Management:
Adding Contacts: Users can add new contacts to their phone book. Each contact will have fields for first name, telephone number, address, email, and comments.
Searching Contacts: Users can search for specific contacts using criteria such as name, telephone number, or email.
Listing Contacts: Users can view a list of all their contacts, displaying the relevant details for each contact.
Note-Taking:
Adding Notes: Users can add notes for each contact. Notes can be either topological or simple.
Simple Notes: Users can add simple notes with fields for date, title, and note content.
Topological Notes: Users can add topological notes with fields for date, title, note content, and a point on a map (latitude and longitude coordinates).
Associating Notes with Contacts: Users can link notes to specific contacts in their phone book. This allows them to organize and categorize their notes based on the associated contact.
User-Specific Environment: Each user will have their own isolated environment within the application. Their contacts and notes will be separate from other users, ensuring privacy and data integrity.
Overall, this Java web application provides users with the ability to manage their contacts, search for specific contacts, and keep notes associated with each contact. The application's user registration and authentication features ensure secure access and data separation for each user.
